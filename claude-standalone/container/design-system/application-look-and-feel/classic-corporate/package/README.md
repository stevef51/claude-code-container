# @design/application-look-and-feel-classic-corporate

> Design system package generated from **Application look and feel**, variant V1: Classic Corporate

Style reference: **Application Design** — Classic Corporate

## Install

```bash
npm install file:./packages/@design/application-look-and-feel-classic-corporate
```

## Usage

### Tailwind v4 (recommended)

Import the Tailwind preset in your main CSS file:

```css
@import "tailwindcss";
@import "@design/application-look-and-feel-classic-corporate/tailwind-preset.css";
```

### CSS Custom Properties

Import the theme CSS directly:

```css
@import "@design/application-look-and-feel-classic-corporate/theme.css";
```

### JavaScript / TypeScript

Import tokens programmatically:

```typescript
import tokens from "@design/application-look-and-feel-classic-corporate/tokens.json";

// Access colors
console.log(tokens.colors.primary.css); // => "hsl(240 5.9% 10%)"
```
